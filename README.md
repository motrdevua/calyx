Фокус дня Люди обожают использовать новую вкладку как напоминалку. Как сделать круто: Небольшой аккуратный виджет в нижнем углу или по центру. Пользователь вписывает: "Главная задача на сегодня: дописать код". И каждый раз, открывая новую вкладку, он видит эту надпись. Галочку нажал — текст зачеркнулся и эффектно исчез.

правильнуая архитектура слоев:

* **Слой 0-10:** Обои и сетка ссылок.
* **Слой 20-50:** Виджеты рабочего стола.
* **Слой 100:** Плавающие кнопки (шестеренка настроек).
* **Слой 5000+:** Модальные окна и их оверлеи.
